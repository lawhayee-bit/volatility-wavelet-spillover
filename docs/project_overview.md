# 项目概述

## 项目背景

本项目围绕一篇面向 `MDPI Applied Sciences` 的研究型论文展开，研究主题为：

**基于多尺度小波特征与混合机器学习的股指波动率预测与风险预警**

项目定位强调以下几个原则：

- 应用型而非纯金融理论型
- 使用公开数据而非私有数据库
- 方法可复现、可解释、不过度复杂
- 强调滚动窗口评估和强基线比较

项目的核心目标是构建一个能够同时支持**连续波动率预测**与**高风险状态预警**的统一框架，并使其在公开可获得的数据条件下可直接复现。

## 研究问题

项目主要围绕两个相关任务展开：

### 1. 波动率预测

在给定历史价格、技术指标和宏观风险信息的条件下，预测未来若干 horizon 下的股指波动率水平。

### 2. 风险预警

将连续波动率预测进一步映射为高风险状态识别问题，从而生成风险概率、预警标签和潜在的预警等级。

这两个任务共同构成了“forecast-to-warning”的研究框架，即从预测到预警的一体化设计。

## 研究对象与数据范围

项目选取美国三个核心股票指数作为主要研究对象：

- `S&P 500`
- `Nasdaq-100`
- `DJIA`

数据来源以公开、稳定、可核验的平台为主，包括：

- `Stooq`
- `FRED`

所使用的数据类型包括：

- 指数 `OHLC` 数据
- ETF 流动性代理
- 隐含波动率指标
- 利率与期限利差
- 金融条件变量
- 扩展公共风险变量，如信用利差、政策不确定性和波动率期限结构指标

## 目标变量设计

项目以基于 `OHLC` 的波动率代理作为核心预测目标，主方案采用：

- `Rogers–Satchell volatility`

稳健性部分进一步考虑：

- `Parkinson volatility`

在预测任务中，目标不是仅预测单日波动，而是预测未来若干 horizon 的平均波动率，主要包括：

- `h = 1`
- `h = 5`
- `h = 10`

## 方法框架

项目的方法设计建立在“多尺度表示 + 混合预测 + 预警映射”的三层逻辑上。

### 1. 多尺度表示层

对关键时间序列进行因果式非降采样小波分解，提取不同时间尺度上的信号特征。主要输入序列包括：

- 波动率目标序列
- 绝对收益序列
- 隐含波动率代理序列

从小波分解结果中提取的 summary features 包括：

- recent coefficient
- rolling mean
- rolling std
- local energy

### 2. 混合预测层

在传统持久性结构基础上引入非线性机器学习模型，构成混合预测框架。主要思想包括：

- 使用 `HAR` 类特征保留强持久性基线
- 使用 `LightGBM` 建模非线性交互关系
- 构建 `wavelet_lightgbm` 模型，将原始风险特征与多尺度特征联合输入
- 通过面向 `QLIKE` 的输出校准提高正值波动率预测的稳定性

### 3. 风险预警层

基于滚动窗口中的高分位阈值定义高风险状态，并通过可解释的 warning 模型输出：

- warning probability
- high-risk label

这一层使得模型不只是输出连续预测值，还能支持风险监测和决策辅助。

## 特征工程

项目的特征设计由四类信息组成：

### 1. 持久性特征

- 当前波动率
- 5 日平均波动率
- 22 日平均波动率

### 2. 市场与技术特征

- 收益率
- 隔夜跳跃
- `ATR`
- `RSI`
- `MACD`

### 3. 宏观与风险特征

- `VIX` 类指标
- 利率
- 期限利差
- 金融条件
- 信用利差
- 政策不确定性

### 4. 小波多尺度特征

- target variance multiscale features
- absolute return multiscale features
- implied volatility multiscale features

## 实验设计

项目采用严格的时序实验协议，以避免信息泄漏并保证外样本评估有效性。

### 1. 评估协议

- `rolling-window / walk-forward evaluation`
- 固定长度训练窗
- 每次仅使用历史可得信息训练与预测
- 所有标准化、阈值和校准均在训练窗内完成

### 2. 主比较模型

实验包含多个层级的基线与主模型，例如：

- `HAR`
- `HV-22`
- `LightGBM`
- `Wavelet-LightGBM`

### 3. 回归指标

波动率预测任务重点使用：

- `QLIKE`
- `RMSE`
- `MAE`
- `R^2_{OOS}`

### 4. 预警指标

风险预警任务重点使用：

- `PR-AUC`
- `Brier`
- `ROC-AUC`
- `Precision`
- `Recall`

### 5. 稳健性设计

为检查结论在不同设定下的稳定性，项目进一步加入：

- 更丰富的公共风险变量面板
- `Parkinson` 替代目标
- 机制解释图与特征块贡献分析

## 论文写作定位

项目在论文写作中坚持以下定位：

- 以 `Applied Sciences` 的应用导向为目标
- 强调可复现公共数据框架
- 强调与强基线比较而不是仅追求复杂模型
- 将多尺度方法写成信息组织机制，而不是单纯性能技巧
- 将风险预警作为波动率预测的延伸应用场景

这一定位使论文既保留金融时间序列研究的严谨性，又更贴近应用型机器学习和决策支持系统的期刊要求。
